# Mad Libs LAN Game

This project runs a peer-to-peer Mad Libs game over your local network.

## Raspberry Pi

1. Open a terminal on your Raspberry Pi.
2. Clone the repository:
   ```bash
   git clone git@github.com:miniPCB/Mad_Libs.git
   ```
3. Navigate into the project folder:
   ```bash
   cd Mad_Libs
   ```
4. Run the game:
   ```bash
   python mad_libs.py
   ```

## Desktop (VS Code)

1. Clone the repository:
   ```bash
   git clone git@github.com:miniPCB/Mad_Libs.git
   ```
2. Open VS Code.
3. Open the `Mad_Libs` folder in VS Code.
4. Run `mad_libs.py` from the VS Code terminal:
   ```bash
   python mad_libs.py
   ```

## Notes

- Run the game on two machines on the same local network so they can discover each other.
- If `python` is not found on your system, try:
  ```bash
  python3 mad_libs.py
  ```
